Konceptet: Ni bygger en app där båda kan logga dagliga vanor (träning, sömn, skärmtid, sparande), se varandras progress i realtid, och få veckovisa AI-genererade sammanfattningar med tips via ett LLM-API.

**Vad ni lär er:**

- Python backend med **FastAPI**
- **Användarhantering** — två konton, delad data (viktig databaskoncept)
- **Schemalagda jobb** med APScheduler (skicka veckorapport automatiskt)
- LLM-API för AI-sammanfattningar (låg kostnad, några ören per vecka)
- Notifikationer via **email (SendGrid gratis-tier)**
- Frontend med **React** eller enkel HTML/CSS
- Git/GitHub — perfekt för par, ni jobbar i varsina branches
- Deployment på Railway/Render

**Varför det passar er:** Ni använder det dagligen tillsammans vilket håller motivationen uppe. Schemalagda jobb och email-integration är tekniker som återkommer i nästan alla system. Personlig relevans är hög.
